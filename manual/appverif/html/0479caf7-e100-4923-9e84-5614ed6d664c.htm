A SOCKET handle leak was detected from a layered service provider (LSP)

Application Verifier break message

A SOCKET handle leak was detected from a layered service provider (LSP)

Probable cause

An open SOCKET handle from a layered service provider (LSP) was leaked.

Troubleshooting steps

If Parameter 2 is not NULL, enter dps Parameter 2 to dump the stack where the leaked socket was created.

Information displayed by Application Verifier

Parameter 1 - Handle to the leaked SOCKET

Parameter 2 - Initialization stack trace

Parameter 3 - ID of the thread that opened the handle

Parameter 4 - Not Used

Severity: Error
