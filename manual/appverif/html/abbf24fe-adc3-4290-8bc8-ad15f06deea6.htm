IAppVerifierManager interface
See Also 

Represents the ability to configure Application Verifier. This is the top most interface for Application Verifier and all other interfaces are acquired directly or indirectly from this one.

Methods

IAppVerifierManager::get_BuildVersion method

Retrieves the build version portion of Application Verifier version number.

IAppVerifierManager::get_Checks method

Retrieves the check collection.

IAppVerifierManager::get_Images method

Retrieves the collection of images (applications) that have enabled Application Verifier.

IAppVerifierManager::get_MajorVersion method

Retrieves the major version portion of Application Verifier version number.

IAppVerifierManager::get_MinorVersion method

Retrieves the minor version portion of Application Verifier version number.

IAppVerifierManager::get_PrivateVersion method

Retrieves the private version portion of Application Verifier version number.

IAppVerifierManager::get_Logs method

Retrieves the collection of logs for the specified image.

Inherits from

IDispatch interface
