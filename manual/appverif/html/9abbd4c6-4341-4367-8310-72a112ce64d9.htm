Net Verifier Stops

When Application Verifier is run on a solution or project, the Net Verifier provides a stop code when an error is found. The stop codes that may be encountered depend on the tests selected to run.

Net Verifier test layer messages

These messages are returned by the Net Verifier when it is testing the Net APIs.
