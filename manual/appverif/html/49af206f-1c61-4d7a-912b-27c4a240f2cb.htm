Debugging Verifier Stops

When you enable some of the checks in Application Verifier and save the settings, you may see a message box with the following message:

“To achieve the best result, please test [yourapp.exe] while running it under a debugger. For more information see the “Debugging Verifier Stops” topic in the Application Verifier help file.”

This is an informational message only. You can suppress this message using the checkbox on the message box that says “Don’t show me this message again.”

We recommend you launch your application under a debugger as the information in the log file may not be sufficient for effective troubleshooting.
