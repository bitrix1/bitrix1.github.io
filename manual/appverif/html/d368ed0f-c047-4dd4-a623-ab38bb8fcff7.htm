Stop Codes and Definitions

An Application Verifier stop code is data provided when an error is found by the verification layer that is being run on a project. The stop codes that may be encountered are based on the checks selected to run. Within this topic section, you will find a page per stop code that provides you with the break message, probable cause, information displayed by Application Verifier, and additional information.

How to find the details

There are a few ways to find out the details of a stop code:

  1. I know the stop code: If you know the stop code number (e.g. 000c), you can go directly to the index and click on it. OR you can do a search. Both of these will display the stop code page.

  2. I know the error message: If you know the error message, search with in the help contents to display the page.
