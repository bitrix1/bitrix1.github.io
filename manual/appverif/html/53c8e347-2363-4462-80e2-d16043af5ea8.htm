Single Threaded Object Violation
See Also

Another thread is using a single threaded Web Services Windows API intrinsic object. For more information, read the documentation in the MSDN library.

Application Verifier break message

Multithreaded use of a webservices.dll intrinsic object

Probable cause

An operation in another thread is using a non-thread safe intrinsic Web Services Windows API object.

Troubleshooting steps

To debug this stop:

$ Review the message provided, as it will indicate at least one of the offending APIs.
