Property Window

The Property window provides access to the various settings for each of the verification layers and checks.

Accessing the Property Window

The property window displays its information in the lower half of the screen.

To view the Property window

  • On the View menu, select Property Window . A check mark while appear next to the Property Window selection.

Closing the Property Window

If desired, you can remove the property window from your view.

To close the property window

  • On the View menu, select Property Window . The check mark next to the Property Window selection will be cleared.

Setting Properties

Each verification layer has its own set of properties to set and change. For more details see Setting Properties - 1.

To set a property for a verification layer

  • Select the layer within the test area (for example, Basics). This step will display the properties of the selected test.
