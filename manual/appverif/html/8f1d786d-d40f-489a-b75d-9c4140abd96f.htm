Print driver has mismatch of AddRef/Release calls of PT/PC XML Document

Application Verifier break message

Print driver has mismatch of AddRef/Release calls of PT/PC XML Document

Troubleshooting steps

If Parameter 1 is not NULL, use du Parameter 1 to display the name of the offending method.

If Parameter 2 is not NULL, use du Parameter 2 to display the XML document type.

Information displayed by Application Verifier

Parameter 1 - Name of offending method call.

Parameter 2 – Type of XML Document.

Parameter 3 – Current reference count.

Parameter 4 - Expected reference count.

Additional information

Stop code: D02E
