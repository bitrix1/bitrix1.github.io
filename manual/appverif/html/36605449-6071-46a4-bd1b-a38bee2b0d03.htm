Application Verifier break message

Attempt to use an unknown SOCKET

Probable cause

An attempt was made to use an unknown value for a SOCKET that was not created by a call to Winsock.

Troubleshooting steps

Enter k to dump the stack trace of the function that attempted to use the unknown socket value.

Information displayed by Application Verifier

Parameter 1 - SOCKET being accessed.

Parameter 2 - Not Used

Parameter 3 - Not Used

Parameter 4 - Not Used
