Corrupted verifier TLS structure

Application Verifier break message

Corrupted verifier TLS structure.

Probable cause

This stop is generated if the internal verifier structures used to store the state of a TLS slot for a thread are corrupted. This is very likely due to some random corruption in the process.

Information displayed by Application Verifier

Parameter1 – TEB Address

Parameter2 –.Expected TEB Address

Parameter3 – Thread ID

Parameter4 – Expected Thread ID

Description – Corrupted verifier TLS structure.

Additional information

Verifier stop code 0351.
