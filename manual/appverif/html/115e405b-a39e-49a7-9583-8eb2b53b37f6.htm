Features

Application Verifier has the following features:

Applications

The Applications window contains applications that have been added to Application Verifier to test. Any application you add to this list will continue to be tested each time it is run until it is removed.

To add an application

  1. On the File menu, select Add Application .

  2. Browse to the application and click Open .

or

  1. Right click within the Applications pane and select Add Application .

  2. Browse to the application and click Open .

To remove an application

  1. Highlight the application to be deleted.

  2. On the File menu, select Delete Application .

or

  • Right click on the application and select Delete Application .

Tests

The information within the Tests window shows the verification layers that can be turned on or off for the selected application. By expanding the verification layer, specific tests (also known as checks) are displayed.

To turn on a test

  • Select the checkbox next to the test (check).

To turn on an entire verification layer

  • Select the checkbox at the top level.

To turn a test

Clear the box next the test.

To return to the default settings

  • On the File menu, select Restore Defaults .

See Testing Applications for more details.

Properties

The Properties window displays properties for the selected test.

To view the Properties window

  • On the View menu, click Property Window .

To display the properties for a test

  • Select the name of the test in the Tests window.

Refer to Property Window and Setting Properties for Verification Layers to learn more about this topic.

Logs

Logs are generated when an application is run and errors were encountered based on the tests selected.

To view the available logs

  • On the View menu, select Logs .

Refer to Logs to find out what other options are available for this feature.
