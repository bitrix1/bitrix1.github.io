Application Verifier break message

Unclosed window belonged to the current thread.

Probable cause

This stop is generated if any window is kept live when this threadpool thread is returned to the pool.

Parameter1 - Callback function

Parameter2 - Context

Parameter3 - Threadpool Object allocation stack trace, use dds to dump it.

Parameter4 – Threadpool thread ID.
