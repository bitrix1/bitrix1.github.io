Application Verifier break message

Core driver is sending an unknown printer handle to the plug-in

Probable cause

The core driver is sending the plug-in a printer handle that was not opened by calling OpenPrinterA, OpenPrinterW, OpenPrinter2W in Microsoft Windows Vista, AddPrinterA, or AddPrinterW.

Report this error to Microsoft because it could be a problem in Microsoft's core printer driver module.
