Support

Only the latest release of Application Verifier is supported. As a result, if you feel that you have run into a bug with the tool, please download the latest release from the download center to see if the problem still occurs.

If you have found a problem with Application Verifier and you think it may be a bug, please notify the following newsgroup: microsoft.public.win32.programmer.tools.

If you are having problems identifying an error that was logged, please review the help file for the specific stop code. If you are still unable to figure out the problem, contact the newsgroup.
