Application Verifier break message

A layered service provider (LSP) attempt to use an unknown SOCKET

Probable cause

An unknown SOCKET handle value was used by a Winsock layered service provider (LSP). This is generally indicates to a fault in a specific LSP layered between the application and Winsock.

Troubleshooting steps

Enter k to view the current call stack for the module making the call with the unknown SOCKET value.

Information displayed by Application Verifier

Parameter 2 - Not Used
