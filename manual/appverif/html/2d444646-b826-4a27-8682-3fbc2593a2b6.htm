VerifierErrorReporting enumeration
See Also 

Controls how Application Verifier interacts with a debugger.

 Copy Code
typedef enum 
    VerifierErrorReporting
  
{
    VerifierErrorReportingNoBreak,
    VerifierErrorReportingException,
    VerifierErrorReportingBreakpoint
}

Elements

VerifierErrorReportingNoBreak

When a stop is encountered, Application Verifier will not break into the debugger.

VerifierErrorReportingException

When a stop is encountered, Application Verifier will generate an exception.

VerifierErrorReportingBreakpoint

When a stop is encountered, Application Verifier will break into the debugger.
