The parameters for the timer object are inconsistent

.

Application Verifier break message

The parameters for the timer object are inconsistent. Period should be 0 when WT_EXECUTEONLYONCE is specified when creating the timer.

Probable cause

Application Verifier generates this stop if the parameters for the timer object are inconsistent. The Period to signal the timer is not zero when the timer is set to signal only once with the WT_EXECUTEONLYONCE flag.

Information displayed by Application Verifier

The parameters for the timer object are inconsistent. Period should be 0 when WT_EXECUTEONLYONCE is specified when creating the timer.

Parameter 1 – Period specified

Parameter 2 – Flags specified

Parameter 3 – Not used

Parameter 4 – Not used

Additional information

Verifier stop code 709.
